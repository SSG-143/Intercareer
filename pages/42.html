Frontend Developer Intern

StockTicker


Work From Home • Internship • 2 Months

About the Opportunity


This is a valuable opportunity for students or fresh graduates passionate about web development and design to gain hands-on experience as a Frontend Developer Intern. The internship provides exposure to real-world projects, team collaboration, and modern frontend technologies. Interns will work closely with the product and backend teams to create responsive, user-friendly, and visually appealing web applications. This role helps interns enhance their skills in HTML, CSS, JavaScript, and frontend frameworks like React, Angular, or Vue.js. It's a perfect stepping stone to enter the software development industry, with the potential for full-time placement based on performance.


Responsibilities of the Intern


  • Develop user-facing features using modern frontend technologies like React.js, Vue.js, or Angular
  • Collaborate with UI/UX designers to convert designs into interactive web pages
  • Ensure the technical feasibility of UI/UX designs and optimize applications for speed and scalability
  • Integrate frontend code with backend APIs
  • Write clean, well-documented, and reusable code
  • Test and debug issues to improve overall performance
  • Participate in daily stand-up meetings and report on progress and blockers
  • Work with Git/GitHub for version control and team collaboration

Requirements


  • Currently pursuing a Bachelor's degree in Computer Science, Information Technology, or a related field
  • Solid understanding of HTML5, CSS3, and JavaScript
  • Familiarity with at least one frontend framework/library (React, Angular, or Vue)
  • Basic knowledge of RESTful APIs and asynchronous requests (AJAX, Fetch)
  • Understanding of responsive design and cross-browser compatibility
  • Ability to work independently as well as in a team environment
  • Good communication skills and a willingness to learn
  • Experience with tools like Git, Figma, and Visual Studio Code is a plus

Internship Details


  • Application Deadline: 21 Aug 25, 12:00 AM IST
  • Duration: 2 Months
  • Stipend: ₹10,000 /Month
  • Working Days: 5 Days a Week
  • Type: Work From Home (Full Time)

Perks


  • Job Offer (based on performance)
  • Certificate of Completion
  • Letter of Recommendation
  • Flexible Hours

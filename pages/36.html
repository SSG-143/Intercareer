Web Development Intern

Remote • Internship • 3 Months

About the Internship


TechSolid Development is looking for a passionate Web Development Intern to work on full-stack projects. This internship provides hands-on experience with HTML, CSS, and React.js, offering an excellent opportunity for freshers to learn in a supportive, collaborative environment.


Key Responsibilities


  • Develop and maintain web applications using HTML, CSS, and React.js.
  • Collaborate with senior developers to design and implement new features.
  • Write clean, efficient, and well-documented code.
  • Participate in code reviews and improve code quality.
  • Troubleshoot and debug web applications.

Qualifications & Skills


  • Strong knowledge of HTML and CSS.
  • Experience with React.js and front-end workflows.
  • Basic understanding of Git or version control systems.
  • Good problem-solving skills with a learning mindset.
  • Excellent communication and teamwork abilities.

Internship Details


  • Eligibility: Engineering, MBA, Undergraduate, Postgraduate students, and Freshers.
  • Location: Work From Home
  • Duration: 3 Months
  • Stipend: ₹1,000 – ₹6,000 / Month
  • Working Days: 5 Days a Week
  • Application Deadline: 22 Aug 2025, 12:00 AM IST
  • Perks: Certificate of Completion, Letter of Recommendation

Shopify Front-End Developer

Ecom Experts | Shopify Plus Agency


Remote • Full Time • Monday–Friday, 9 AM–5 PM EST

About the Job


We’re seeking a skilled Shopify Front-End Developer with strong JavaScript and CSS expertise to help build and maintain a high-performing e-commerce platform. In this role, you will work with Shopify’s Liquid framework, DOM manipulation, asynchronous JavaScript, and advanced styling to create a seamless user experience.


Key Responsibilities


  • Develop and customize Shopify themes using Liquid, meta fields, DOM manipulation, and asynchronous JavaScript.
  • Ensure UI/UX designs are technically feasible, optimized for speed, and scalable.
  • Write clean, reusable code and maintain libraries for future projects.
  • Collaborate with developers to meet project deadlines.
  • Stay updated on new trends, tools, and techniques in web development.

Requirements


  • Hands-on experience with Shopify, Liquid, and Shopify Objects.
  • Strong CSS knowledge, including advanced selectors.
  • Proficiency in DOM manipulation and asynchronous JavaScript.
  • Experience with Git/GitHub version control.
  • Knowledge of GraphQL and React.js is a plus.

Qualifications


  • Bachelor’s degree in Computer Science, Software Engineering, or equivalent experience.
  • Excellent communication skills and teamwork abilities.
  • Portfolio showcasing Shopify and front-end development projects.

Preferred Skills


  • Strong knowledge of Shopify’s customization tools, meta fields, and admin panel.
  • Experience with JavaScript frameworks like jQuery or Vue.js.
  • Responsive design and media query implementation.
